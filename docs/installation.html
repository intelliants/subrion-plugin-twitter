Installation is a very simple process. Just click Install and the plugin installation will be performed.

Getting Twitter Widget ID

First, you should create your Twitter Widget. You may ignore such options like Height, Theme, and Link Color as far as they are configured by our script.

Once you created your widget, copy the long number you see in your Browser's address bar:

This number is your Twitter Widget ID. Please put it into Admin Panel > Configuration > Miscellaneous > Twitter Widget ID and save the changes.
